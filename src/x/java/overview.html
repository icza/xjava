Extension and utility classes for the standard Java library.

Other modules of the XJava project also build on this.

For logging the project uses the JavaTM 2 platform's core logging facilities. Each class uses a logger acquired like this:

java.util.logging.Logger.getLogger(<classname>)

For configuring logging (e.g. formatter, log level) you can use a logging.properties file for example. More info about it: Java Logging Overview
