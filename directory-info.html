XJava Project Directory Info

File/folder Description Part of
Release
/src Source code of the core XJava module. ✔
/src-servlet Source code of the Servlet API XJava module. ✔
/src-test JUnit test sources of the project. ✖
/bin Default output folder (for the /src source folder). ✔
/bin-servlet Output folder for the Servlet API module (/src-servlet source folder). ✔
/bin-test Output folder for the /src-test JUnit tests. ✖
/lib Contains the external (API) libraries used by the XJava implementation. ✖
/lib-ant-junit Contains the external libraries required/used by the ant build script to run the JUnit tests. ✖
/release / Root Release folder, release building process uses this as the work folder. ✖
/xjava Base Release folder, contains an assembled release. ✖
/doc Project related documentations. ✖
/eclipse-conf Eclipse configuration files for the project. Required only for developing the project in Eclipse. ✖
/build.xml Ant Build script to run JUnit tests, create releases. ✖
